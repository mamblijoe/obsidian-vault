## Таблицы с прямым доступом

```js
let hashMapDirectAccess = { 
	'key1': 'value1',
	'key2': 'value2',
	'key3': 'value3' 
}; 
	
console.log(hashMapDirectAccess['key1']);
```

## Таблицы с индексированным доступом

```js
let hashMapIndexAccess = { 
	0: 'apple', 
	1: 'orange',
	2: 'banana'
};
	 
console.log(hashMapIndexAccess[0]);
```
