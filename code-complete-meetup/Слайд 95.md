>[!info] Размыкание цикла
```ts
// Хуже
for (let i = 0; i < sales.length; i++) {
  if (type === "net") {
    netSum += sales[i];
  } else if (type === "gross") {
    grossSum += sales[i];
  }
}

// Лучше
if (type === "net") {
  for (let i = 0; i < sales.length; i++) {
    netSum += sales[i];
  }
}

if (type === "gross") {
  for (let i = 0; i < sales.length; i++) {
    netSum += sales[i];
  }
}
```
