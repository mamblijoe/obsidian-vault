>[!info] Вложение более ресурсоемкого цикла в менее ресурсоемкий
```ts
// Хуже (600 итераций)
for (let col = 0; col < 100; col++) {
  for (let row = 0; row < 5; row++) {
    sum = sum + table[row][col];
  }
}

// Лучше (505 итераций)
for (let row = 0; row < 5; row++) {
  for (let col = 0; col < 100; col++) {
    sum = sum + table[row][col];
  }
}
```
