>[!info] Минимизация объема работы в цикле
```ts
// Хуже
for (let i = 0; i < sales.length; i++) {
  netRate[i] = baseRate[i] * rates.discounts.factors.net * 10 - 15;
}

// Лучше
const discount = rates.discounts.factors.net * 10 - 15;
for (let i = 0; i < sales.length; i++) {
  netRate[i] = baseRate[i] * discount;
}
```
