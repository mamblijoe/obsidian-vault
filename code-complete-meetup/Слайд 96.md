>[!info] Объединение цикла
```ts
// Хуже
for (let i = 0; i < sales.length; i++) {
  salesName.push(sales[i].name);
}

for (let i = 0; i < sales.length; i++) {
  salesId.push(sales[i].Id);
}

// Лучше
for (let i = 0; i < sales.length; i++) {
  salesName.push(sales[i].name);
  salesId.push(sales[i].Id);
}
```
