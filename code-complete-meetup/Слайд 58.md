>[!info] Проверяйте зависимости с помощью утверждений или обработки ошибок
```ts
class Calculator {
  public count = 0;

  public compute() {
    if (!!this.count) {
      throw new Error();
    }
    // ...
  }

  public computeCount() {
    this.count = 20;
    // ..
  }
}

const calculator = new Calculator();
calculator.computeCount();
calculator.compute();
```
